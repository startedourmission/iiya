# IIYA Project

## 프로젝트 목적
클로드 코드로 만들 수 있는 **최고로 간지나는 결과물**들을 모아놓는 프로젝트.
비개발자가 딱 보고 "이야~" 할 수 있는 비주얼이 최우선.
코드 퀄리티, 분야 무관. **보여지는 것이 전부.**

## 구조
- 각 작품은 **하위 폴더**를 만들어서 독립적으로 관리
- 루트 `index.html`은 전체 작품 목록/갤러리 역할 (추후)
- 예: `cosmic-journey/index.html`, `next-project/index.html`

## 배포
- GitHub Pages로 배포 (정적 사이트)
- 백엔드 없이 프론트엔드만으로 구성

## 규칙
- 백엔드 없이 GitHub Pages 배포 가능한 형태로 만들 것
- 백엔드가 필요한 경우 반드시 사용자에게 검토 요청할 것
- 서버 사이드 로직 추가 금지 (사전 승인 없이)
- 새 작품은 항상 하위 폴더를 만들어서 작업할 것
