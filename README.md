# IIYA ✦

> 비개발자가 보고 "이야~" 할 수 있는 것들만 모았습니다.

클로드 코드로 만든 비주얼 실험 모음집.
코드 퀄리티, 분야 무관. **보여지는 것이 전부.**

<br>

## Works

| # | Project | Description |
|---|---------|-------------|
| 01 | [**Cosmic Journey**](./cosmic-journey) | 인터랙티브 우주 시뮬레이션 — 셰이더 기반 성운, 별, 오로라 |
| 02 | **Fluid Sim** | 마우스로 만지는 유체 시뮬레이션 — 물감이 퍼지듯 색이 섞이는 |
| 03 | **Audio Visualizer** | 음악에 반응하는 3D 파형 비주얼라이저 |
| 04 | **Particle Life** | 입자들이 끌고 밀며 생명체처럼 군집하는 인공생명 |
| 05 | **Terrain Generator** | 절차적 3D 지형 생성 — 산, 바다, 숲, 날씨 |
| 06 | **Infinite Zoom** | 프랙탈 기반 무한 줌 — 끝없는 만델브로 집합 |
| 07 | **Gravity Playground** | 행성을 놓으면 중력 궤도가 형성되는 태양계 시뮬레이터 |
| 08 | **Neon City** | 사이버펑크 3D 도시 자동 생성 플라이스루 |
| 09 | **Silk Weaver** | 마우스 궤적을 따라 실크 곡선이 엮이는 제너러티브 아트 |
| 10 | **Wave Interference** | 파동 간섭 시뮬레이션 — 파문과 색의 간섭 패턴 |

<br>

## Stack

- HTML + CSS + JavaScript + WebGL
- 백엔드 없음. 전부 브라우저에서 돌아감.
- GitHub Pages 배포

<br>

## How to Run

아무 프로젝트 폴더의 `index.html`을 브라우저에서 열면 끝.
